# Car Rental Portal Management System

A Django-based Car Rental Portal Management System that allows users to browse, book, and manage rental cars.

## Features

- User Registration and Login
- Car Booking System
- Car Availability Management
- Booking Management
- Admin Panel
- Responsive User Interface

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
- JavaScript

## Installation

1. Clone the repository.
2. Install requirements.
3. Run the Django server.
